模块描述

三相逆变模块是一块仅需少量、廉价且简单的电子元件，即可实现将直流电转化为三相电的功能。并且能够稳定承受100W的功率。模块还具有过载保护功能。

功能特性

模块具备以下功能特性：

  • 模块支持PWM、SPWM等控制信号。可单独控制每一个开关管
  • 具有过载保护功能。
  • 上下桥臂内置死区，无需担心短路问题。
  • MOSFIT管基极具有电荷释放电路，防止电荷积累击穿MOS管。
  • 驱动电阻方便更换调整。
  • 使用电阻更小的香蕉头接线柱。
  • 可设置外部程序使能，提供手动保护功能。
  • 留有RCD吸收回路，以备应急使用。
  • 留有大量测试点，方便调试。

电路分析与注意事项

三桥臂电路

电路下图所示：

其中R19～R24是较为重要的元件，电阻的作用是在基极状态变换时缓解电压突变、而电流无法突变时，对MOSFIT产生的巨大冲击。其阻值微小的变化就可能对输出波形的尖峰与振铃带来可观的缓解。

{% note info %}
注意：
本模块中这些电阻的大小建议使用50欧姆以上。如果您使用20欧姆，操作不当，很有可能导致驱动器损坏。
{% endnote %}

另外也可以通过在直流电对地并联大容量电解电容、高频陶瓷电容缓解电压中的不稳定因素。

ITRIP接口是电流采样接口，R27是采样电阻，阻值选用0.025欧姆。

驱动控制电路

电路图如下：

此电路中，C9～C11自举电容的作用将成为理解电路的关键。自举电容就是在低端管开通时充电积攒电压；在高端管将要开通时，将积攒的电压叠加在源极上传输给基极，实现高端管的开通。此电容的容值不必很大，使用钽电容也可达到同样的效果。

管脚10为使能端，已通过排针接出。

软件支持

您可以使用各类MCU、FPGA，生成驱动能力较强的PWM、SPWM波形，为模块提供控制信号。建议添加光耦隔离对主控进行保护。

作者使用STM32F407最小系统板（已开源），利用内置时钟直接生成PWM与SPWM波供给模块，亦可应付大部分应用场景。
