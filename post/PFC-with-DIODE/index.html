模块描述

硬件控制的二极管功率因数校正模块使用ICE1PCS01功率因数校正芯片控制的Boost升压电路。电感电流为CCM连续模式。模块最高能达到0.98的校正数值。

此模块应串接在整流桥之后，为整流后的交流点进行PFC校正。

功能特性

模块具备以下功能特性：

  • 模块无需程序控制，自动进行校正。
  • 具有过载保护功能。
  • 可通过电位器调整模块的功率。
  • 使用电阻更小的香蕉头接线柱。
  • 电路简单易懂。

电路分析与注意事项

升压电路

电路下图所示：

这部分电路是一个标准的Boost电路，其中尤其注意的是R3为电流检测电阻，阻值不易太大，稳定在1欧姆以内，推荐使用水泥电阻，此电阻的大小对电流波形的采样影响巨大，应慎重选择。

{% note info %}
注意：
图中IN+与IN-为整流桥生成的馒头波形的直流电，请勿接入交流点。
{% endnote %}

输入端C1应使用高频陶瓷电容，对谐波过滤，避免使用电解电容，破坏输入电压与电流的波形。

VSENSE采样点应采取稳定的输出电压值，其电路布局应在电容稳压之后。

驱动控制电路

电路图如下：

R4应特别注意阻值的选用，其作用是限制ISENSE端的流入电流，避免烧坏芯片。

R1为驱动电阻，根据MOSFIT管的不同选取不同的阻值。

R2为开关频率调整电阻，查看芯片的datasheet获取其选用方法。
