开发板描述

通用STM32F407最小系统开发板是作者面向电设竞赛自主设计的一款最小系统板，历经了三代进化，从一代的F1到三代的F4。

功能特性

模块具备以下功能特性：

  • 最小系统将可用接口全部引出，方便转接与二次开发。
  • 最小系统留有一个状态灯、ADC输入以及部分按钮，方便简单开发。
  • 最小系统板载Jlink接口、串口、USB供电接口与I2C接口。
  • 最小系统支持3.3v与5v以上供电，自带电压转换。
  • 板载7Pin-OLED接口，可接入屏幕。

{% note info %}
注意：
5V一下只接受3.3V供电。
{% endnote %}

元件需求

本模块需要以下芯片与晶体管（常用电子元件以及封装请查看工程文档）：

  • STM32F407VGT6（相同封装的F407亦可）
  • AMS1117-3.3

软件支持

您可以使用Keil（Win）、VB（Win、Mac）等ARM嵌入式开发软件进行编程开发，通过Jlink或外置USB串口下载工具进行下载与调试。

在开源PCB文件中，您可以查看引脚的排列和其他信息。

版本更新
