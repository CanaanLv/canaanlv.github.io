模块描述

真有效值RMS转换模块使用互感器、RMS计算器、运放组成，模块具有输入隔离效果，其高精度可应用于大功率与小信号等多种场合。

功能特性

模块具备以下功能特性：

  • 差分输入的真有效值RMS转换芯片。
  • 输入具有隔离功能。
  • 模块各个器件的缩小与放大比例均可调。
  • 输出配有放大器，提高采样宽度。

{% note info %}
注意：
此模块需要另外的5V与12V的芯片供电电压
{% endnote %}
